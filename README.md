# projectTerapy
Arquivos para Projeto Java que irá trabalhar com os atendimentos de terapia 

O projeto está sendo desenvolvido em Java
Tem por objetivos
1) O acompamnhamento dos atendimentos prestados por Assistentes Terapêuticas (ATs) aos pacientes com TEA
2) Cadastro de Pacientes
3) Cadastro de Profissionais
3.1) Assistentes Terapêuticas
3.2) Supervisores
4) Cadastro de dias e horários de atendimento
5) Controle de dias e horários de atendimento
6) Emissão de Relatórios de Atendimentos
